Checklist Sains Komputer SPM 2025

Bahagian A

  • 1. Teknik Pemikiran – ciri-ciri
  • 2. Simbol berkaitan dengan ERD / Carta alir / operator
  • 3. Analisis Terbaik Pengaturcaraan
  • 4. Ciri-ciri pembolehubah, pemalar
  • 5. Kenal pasti pembolehubah, pemalar
  • 6. Hitung dalam atur cara (bagi atur cara / suruh hitung purata / guna operator betul)
  • 7. Kenal pasti input/output dalam atur cara (Java vs JavaScript)
  • 8. Simbol-simbol berkaitan ERD, Get logik, carta alir
  • 9. Objek dalam pangkalan data (Form, Report, Table, Query)
  • 10. Model-model dalam Pangkalan Data (Tulis model / contoh / ciri-ciri)
  • 11. Prinsip Reka Bentuk Interaksi
  • 12. Penyulitan
  • 13. Komponen dalam Hardware Von Neumann (ALU, Daftar, CPU)
  • 14. Get Logik (Namakan atau lukis, Ungkapan Boolean)
  • 15. Bahasa Penskripan Pelayan dan Pelanggan (Fungsi dan contoh, Pelayan / Pelanggan – HTML, CSS)
  • 16. CSS – tanya fungsi
  • 17. SQL – coding (SELECT, MIN, SUM, MAX, Fungsi – memformat dokumen)
  • 18. Prinsip antara muka
  • 19. Isihan – ciri-ciri, dua coding (Hafal dan susun)

Bahagian B

  • 21. Get logik – agak susah (Start daripada situasi, Lukiskan, Diberi kebenaran, Exclusive OR – hafal)
  • 22. SQL – Bina jadual, SELECT sum, average, min, max, CREATE TABLE, Manipulasi data
  • 23. Carta Alir – Bentuk antaramuka atau pseudokod, Tumpu coding isihan
  • 24. Prinsip Reka Bentuk Interaksi – Rujuk soalan Kelantan Trial, Elemen KBAT, Buat ayat penilaian sendiri, Buat perbandingan

Status checklist akan disimpan dalam pelayar anda (localStorage).
